Flask

Flask is an "Event Management Application". It gives information about the event's date, time, it's venue. It allows user to add new events, edit the existing events. It gives information about parking area near to the event's venue, Nearby bars and restaurants, traffic information, provides venue map, facility to book tickets.
This application also includes the information about tailgate parties. This section gives information about tailgate party's  venue. User can add it's own tailgate and user can edit the tailgate. User can add the tailgaters for the party. Owner of the tailgate can add the supply list for the tailgate party and can assign supplies items to tailgaters.
This application's third section is friends section. Here user can invite or block the users. User can create group and can add the friends to that group. User can send the messages to his/her friend.
